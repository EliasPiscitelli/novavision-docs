## Proceso comercial Enterprise

### Paso 1: Consulta (lead califica)
El cliente llega por: landing ("Consultá por Enterprise"), referido, o contacto directo.
Se agenda una call de 30 min.

### Paso 2: Discovery (1-2 reuniones)
- ¿Qué vendés?
- ¿Qué catálogo tenés? (volumen, variantes, complejidad)
- ¿Qué integraciones necesitás? (ERP, contabilidad, logística, marketing)
- ¿Tenés identidad visual? (logo, paleta, fotos)
- ¿Qué funcionalidades son críticas vs nice-to-have?

### Paso 3: Propuesta (documento cerrado)
Propuesta con:
- Alcance del setup (qué se hace, qué no)
- Precio del setup
- Precio de la suscripción mensual
- Timeline estimado (típico: 2-4 semanas)
- Extras si aplican (con cotización separada)

### Paso 4: Firma + anticipo
- 50% del setup al firmar
- 50% del setup al entregar
- Suscripción mensual comienza al publicar la tienda

### Paso 5: Implementación (2-4 semanas)
Discovery → Config → Migración → Adaptación → QA → Capacitación → Publicación

### Paso 6: Operación continua
Suscripción mensual. Soporte premium. Mejoras menores incluidas. Extras cotizados aparte.

## Cuándo vender Enterprise y cuándo no

### Vender Enterprise cuando:
- El cliente tiene catálogo grande (+500 productos) o complejo (muchas variantes)
- Necesita integración con sistema existente (ERP, contabilidad)
- Quiere acompañamiento en la implementación
- Tiene presupuesto para setup + mensualidad
- Es una empresa con decisor de compra no técnico que necesita "que le resuelvan"

### NO vender Enterprise cuando:
- El cliente solo quiere "algo más barato" — eso es Starter
- El pedido es "haceme un software completo custom" — eso es una software factory, no NovaVision
- El alcance se corre todo el tiempo sin cierre — red flag de cliente tóxico
- Lo que pide requiere romper el modelo multi-tenant por capricho — no vale la pena

## Función financiera de Enterprise

Para NovaVision como negocio autofinanciado:

| Función | Cómo |
|---------|------|
| **Caja inmediata** | Setup de USD 500-2.000 es revenue day-1 |
| **MRR premium** | USD 390/mes es 6.5x Starter y 19.5x más margen absoluto |
| **Financiar roadmap** | Extras cotizados aparte financian desarrollo que mejora el core |
| **Validar mercado** | Los pedidos Enterprise muestran qué necesita el mercado real |
| **Prueba social** | "Empresas como X usan NovaVision" es el mejor argumento para pymes |

**Pero con límites:**
- No más de 2-3 Enterprise activos simultáneos como founder solo
- Cada Enterprise nuevo no puede secuestrar el roadmap del producto
- Si un Enterprise pide algo que rompe multi-tenant, se rechaza
